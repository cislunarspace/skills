# Research Ethics

帮助用户理解研究伦理，确保学术诚信，包括避免抄袭、正确引用和保持诚信。

## 步骤

### 1. 理解研究伦理

进行研究与撰写论文是一种具有伦理意涵的社会活动：

- **诚实原则**：诚实地报告研究结果，不伪造或篡改数据
- **尊重原创**：尊重他人的知识产权和学术贡献
- **责任担当**：对自己的研究行为负责

### 2. 避免抄袭

抄袭是使用他人的文字或想法却没有注明原作者：

- **文字抄袭**：一字不漏地使用某笔数据的语句
- **想法抄袭**：使用别人的想法却没有注明原创者
- **间接抄袭**：概述或改写原始资料时过于贴近原文

### 3. 正确引用

正确引用是学术诚信的基本要求：

- **直接引文**：使用引号，精确誉录，注明出处
- **改写概述**：用自己的理解重新过滤，注明出处
- **参考文献**：记录完整的参考文献数据

### 4. 保持诚信

研究者应该保持学术诚信：

- **承认限制**：承认证据的限制和研究的局限
- **回应质疑**：诚实回应读者的质疑和反对
- **持续改进**：接受批评，不断改进研究

## 边界情况

| 情况 | 处理方式 |
|------|----------|
| 不确定是否抄袭 | 咨询老师，尽量注明出处 |
| 引用格式不正确 | 查阅引用指南，使用标准格式 |
| 伦理困境 | 咨询导师或伦理委员会 |
| 证据不可靠 | 寻找更可靠的来源，承认证据限制 |
| 发现错误 | 主动承认，及时修正 |

## Checkpoint

- 在用户引用资料时，确认引用是否正确
- 在用户改写概述时，确认是否避免了抄袭
- 在用户报告结果时，确认是否诚实准确

## 完成条件

- 用户理解研究伦理的重要性
- 用户能够避免各种形式的抄袭
- 用户知道如何正确引用资料
- 用户能够保持学术诚信
