# Source Collection

帮助用户寻找可靠的原始资料，包括图书馆资源、互联网资源和直接搜集资料。

## 步骤

### 1. 筛选可靠的原始资料

原始资料分为三种：
- **初级原始资料**：你正研究的那个时期或作者所写的原典
- **二级原始资料**：以原始资料为基础的研究报告
- **三级原始资料**：综合及解说某个领域的研究

可靠的原始资料特征：
- 由信誉良好的出版社出版
- 经过该领域专家的审查
- 作者是有名望的学者
- 资料是最新的

### 2. 使用图书馆资源

图书馆是研究的首站：

- **咨询图书馆管理员**：他们可以告诉你如何使用在线目录
- **一般参考书籍**：从《大英百科全书》等开始
- **专业参考书籍**：查找特定领域的参考文献资源
- **研究指南**：找到资深研究者惯用的原始资料
- **馆藏查询**：通过关键词搜索和浏览扩展查询

### 3. 使用互联网资源

互联网的优点通常也是它的缺点：没有把关者。

可靠的互联网资源：
- 由可靠的期刊或出版社所提供的信息
- 与图书馆找到的形式完全相同
- 增补纸质的原始资料
- 只有在互联网上才找得到的数据
- 作为你的初级原始资料

### 4. 直接搜集资料

与民众谈话才能获取数据：

- **作为参考文献来源的专家**：老师、同学、亲戚、朋友
- **作为初级原始资料的民众**：通过访谈搜集初级资料
- **伦理准则**：遵循"不伤害原则"，获得必要许可

## 边界情况

| 情况 | 处理方式 |
|------|----------|
| 找不到资料 | 咨询图书馆管理员，尝试不同关键词，扩大搜索范围 |
| 资料不可靠 | 检查出版社、作者、审查机制，寻找更可靠的来源 |
| 资料太多 | 缩小题目范围，聚焦特定问题，筛选最相关的资料 |
| 资料太旧 | 寻找最新研究，检查引用的论文日期 |
| 互联网资料不可靠 | 优先使用纸质资料，验证互联网来源的可信度 |

## Checkpoint

- 在用户确定资料来源后，确认资料是否可靠
- 在用户搜集资料后，确认资料是否足够且具有代表性
- 在用户使用互联网资料时，确认来源是否可信

## 完成条件

- 用户了解原始资料的三种类型
- 用户能够筛选可靠的原始资料
- 用户知道如何使用图书馆和互联网资源
- 用户了解直接搜集资料的方法和伦理
