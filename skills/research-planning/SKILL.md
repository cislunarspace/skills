# Research Planning

帮助用户理解研究的意义，建立与读者的联系，规划研究过程。

## 步骤

### 1. 理解研究的意义

研究是世界上最古老的对话之一。每个人都在做研究：搜集资料，回答问题以解决难题。研究的价值在于：

- **学习技巧**：研究技巧对任何领域的高级工作至关重要
- **批判思维**：只有亲身经历研究的不确定性，才会彻底评价所读的文章
- **知识贡献**：研究可以改善我们生活的每一层面

### 2. 与读者建立联系

研究如果没有读者阅读，将毫无价值。作者必须为读者创造角色：

- **角色一**：分享有趣的事物（让读者愉悦）
- **角色二**：解决实际难题（提供实用建议）
- **角色三**：回答重要问题（帮助读者更了解事物）

学术研究中最常见的是第三种角色：帮助读者更了解某些事物。

### 3. 研究规划

有经验的研究者都知道，当他们有个计划时，研究总会稳健切实地逐步进行：

1. **找到题目**：从兴趣开始，缩小到可处理的范围
2. **提出问题**：用提问来质疑题目，发现值得探究的问题
3. **寻找资料**：确定读者期待的证据类型，找到可靠的原始资料
4. **构建论证**：提出观点，用理由和证据支持

### 4. 心态准备

所有研究者都是从初学者开始的。处理复杂研究的最好方法：

- **接受焦虑**：不确定感和焦虑是无法避免的，这是缺乏经验的现象
- **随时书写**：写下概述、评论、问题，写下的东西越多，越有信心
- **分步进行**：把整个过程分成可处理的数个步骤去了解整体
- **寻求帮助**：期待老师能了解你的努力，好老师希望你能成功

## 边界情况

| 情况 | 处理方式 |
|------|----------|
| 用户对研究缺乏兴趣 | 从用户感兴趣的领域开始，寻找相关研究题目 |
| 不知道如何开始 | 从图书馆或互联网浏览参考书目，寻找灵感 |
| 对研究感到焦虑 | 说明焦虑是正常现象，提供分步指导 |
| 题目太宽泛 | 使用缩小技巧：增加特定文字、限定范围、聚焦问题 |
| 找不到资料 | 咨询图书馆管理员，使用专业数据库，尝试不同关键词 |

## Checkpoint

- 在用户理解研究意义后，确认他们是否找到感兴趣的领域
- 在用户确定题目后，确认题目是否足够具体
- 在用户提出问题后，确认问题是否值得探究
- 在用户开始研究前，确认他们是否准备好心态

## 完成条件

- 用户理解研究的意义和价值
- 用户确定了研究题目和问题
- 用户了解研究规划的基本步骤
- 用户对研究过程有信心
